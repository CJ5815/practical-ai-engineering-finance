# Practical AI Engineering for Finance

This course teaches a college senior how to build useful AI applications rather
than merely list introductory AI certificates on a résumé.

## What You Will Build

The semester project is an **AI-Powered Equity Research Assistant** that can:

- retrieve public financial information through APIs;
- process and store company documents;
- search documents by meaning;
- answer questions using retrieved evidence;
- include citations or source references;
- evaluate answer quality;
- expose functionality through an API;
- run locally and be deployed for demonstration.

## Weekly Commitment

The course requires **four one-hour sessions each week**:

| Session | Main purpose |
|---|---|
| Day 1 | Learn the concept |
| Day 2 | Follow a guided coding exercise |
| Day 3 | Build independently |
| Day 4 | Test, document, and reflect |

## Evidence of Learning

At the end of the course, the student should have:

- a public GitHub repository;
- readable Python code;
- automated tests;
- a deployed course project;
- a clear README;
- a short demonstration video or presentation;
- specific examples to discuss during interviews.
